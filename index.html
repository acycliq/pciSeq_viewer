About This Viewer

This web-based viewer renders large spatial transcriptomics datasets with smooth pan/zoom and multi-plane visualization. Use the left drawer to toggle genes and cell classes, and the bottom slider to navigate planes. Charts and tools open as widgets.

Marker size: by default, spots appear larger when they are closer to the current plane and smaller when they are farther away (distance-aware sizing). Use the Size slider in Gene Controls to scale all markers, or enable Uniform Size to keep the same size across all planes.

3D view: a separate voxel-by-voxel perspective viewer is available. It renders volumetric cells/genes with depth, supports slicing, and is useful for inspecting structure in 3D. Launch it via the Selection tool: enable Tools\Selection, then hold Ctrl and drag to draw a rectangle; releasing spawns the 3D view for that region.
